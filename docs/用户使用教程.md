# 用户使用教程

## 软件用途

JCC S17 AI Assistant 是一个金铲铲 S17 星神版本局势分析和策略推荐工具。软件用于手动输入或截图识别当前对局信息，并向后端请求策略建议。

软件只展示建议，不会自动操作游戏。

## 安装客户端

1. 获取 Windows 安装包。
2. 双击安装包。
3. 按安装向导完成安装。
4. 启动 `JCC S17 AI Assistant`。

默认后端地址为：

```text
http://127.0.0.1:8000
```

## 登录账号

1. 打开客户端。
2. 进入“登录”页面。
3. 输入 `username` 和 `password`。
4. 点击“登录”。

登录成功后，客户端会把 token 保存到本地 `localStorage`，后续接口请求会自动使用。

## 激活卡密

1. 进入“卡密”页面。
2. 输入购买或管理员发放的 `license_key`。
3. 点击“激活”。

激活成功后，卡密会绑定到当前登录用户。

## 设置后端地址

1. 进入“设置”页面。
2. 修改“后端地址”。
3. 点击“保存设置”。

本地开发默认地址：

```text
http://127.0.0.1:8000
```

如果后端部署在服务器，请填写服务器地址，例如：

```text
https://api.example.com
```

## 手动输入局势获取推荐

1. 进入“局势”页面。
2. 检查或修改默认局势数据：
   - level
   - gold
   - hp
   - round
   - shop
   - board
   - bench
   - items
   - god_choices
   - selected_gods
   - main_god
   - preferred_model
3. 点击“获取推荐”。
4. 客户端会请求后端 `/api/game/analyze`。

## 截图识别获取推荐

1. 进入“截图”页面。
2. 在“截图来源”中选择游戏窗口或屏幕。
3. 选择 `preferred_model`。
4. 点击“手动截图识别”。
5. 等待 OCR 识别完成。
6. 检查“可编辑 JSON”。
7. 点击“确认并获取推荐”。

客户端会把确认后的 JSON 发送到后端 `/api/game/analyze`。

## 如何校准识别区域

截图页提供“识别区域校准”功能，可为以下字段配置区域：

- level
- gold
- hp
- round
- shop
- board
- bench
- items
- god_choices

每个区域包含：

- x
- y
- width
- height

使用方法：

1. 选择截图来源。
2. 根据游戏画面位置调整各字段区域。
3. 点击“保存区域配置”。
4. 配置会保存到浏览器 `localStorage`。
5. 后续截图识别会按保存区域裁剪并分别 OCR。

如果某个字段置信度较低，页面会标记“建议人工确认”。

## 如何查看 AI 推荐结果

推荐结果页会展示：

- recommendations
- model_used
- estimated_cost_usd
- ai_status
- ai_error

`ai_status` 常见值：

- `success`：外部 AI 调用成功。
- `failed`：外部 AI 调用失败，已使用规则推荐 fallback。
- `disabled`：后端未启用外部 AI，只使用规则推荐。

## 常见问题

### 登录失败

确认后端服务已经启动，并检查账号密码是否正确。

### 卡密激活失败

确认卡密存在、未被其他用户激活，并确认已登录。

### 推荐接口失败

确认后端地址正确，后端 `/health` 接口可访问。

### 截图页没有窗口列表

请使用 Electron 客户端启动，不要只在浏览器里打开 Vite 页面。

### OCR 识别不准

调整识别区域，并在“可编辑 JSON”中人工修正后再获取推荐。

### AI 状态为 disabled

后端 `.env` 中 `ENABLE_AI_RECOMMENDATION=false`，这是正常情况。开启外部 AI 需要管理员配置 API Key。

## 安全说明

本软件只做截图识别和推荐展示：

- 不注入游戏。
- 不读取游戏内存。
- 不自动操作游戏。
- 不修改游戏文件。
- 不控制鼠标键盘执行游戏操作。

用户应遵守游戏和平台规则，自行判断是否适合使用。
