# 管理员部署说明

## 后端启动方式

进入后端目录：

```powershell
cd D:\JCC\jcc-s17-ai\backend
```

创建虚拟环境并安装依赖：

```powershell
python -m venv .venv
.venv\Scripts\python.exe -m pip install -r requirements.txt
```

启动服务：

```powershell
.venv\Scripts\python.exe -m uvicorn main:app --host 127.0.0.1 --port 8000
```

健康检查：

```powershell
curl.exe http://127.0.0.1:8000/health
```

## .env 配置说明

复制示例配置：

```powershell
Copy-Item .env.example .env
```

主要字段：

```env
DEEPSEEK_API_KEY=
DASHSCOPE_API_KEY=
OPENAI_API_KEY=
APIROUTER_API_KEY=
AIPOWER_API_KEY=
AIPOWER_BASE_URL=
DEFAULT_AI_PROVIDER=auto
ENABLE_AI_RECOMMENDATION=false
```

说明：

- `ENABLE_AI_RECOMMENDATION=false`：只使用本地规则推荐。
- `ENABLE_AI_RECOMMENDATION=true`：尝试调用外部 AI。
- `DEFAULT_AI_PROVIDER=auto`：由后端根据局势自动选模型。

## API Key 配置

按供应商填写：

```env
DEEPSEEK_API_KEY=sk-...
DASHSCOPE_API_KEY=sk-...
OPENAI_API_KEY=sk-...
APIROUTER_API_KEY=sk-or-...
AIPOWER_API_KEY=sk-...
AIPOWER_BASE_URL=https://your-aipower-endpoint/v1/chat/completions
```

常用模型选择：

- `preferred_model=auto`
- `preferred_model=deepseek`
- `preferred_model=qwen`
- `preferred_model=openai`
- `preferred_model=apirouter`
- `preferred_model=aipower`

## 创建测试用户

启动后端后执行：

```powershell
curl.exe -X POST http://127.0.0.1:8000/register -H "Content-Type: application/json" -d "{\"username\":\"test\",\"email\":\"test@example.com\",\"password\":\"123456\"}"
```

登录测试：

```powershell
curl.exe -X POST http://127.0.0.1:8000/login -H "Content-Type: application/json" -d "{\"username\":\"test\",\"password\":\"123456\"}"
```

## 创建测试卡密

进入后端目录执行：

```powershell
.venv\Scripts\python.exe -c "from app.database import Base, engine, SessionLocal; from app.models import License; Base.metadata.create_all(bind=engine); db=SessionLocal(); key='TEST-KEY-001'; exists=db.query(License).filter(License.license_key==key).first(); db.add(License(license_key=key, duration_days=30, status='unused', device_limit=1)) if not exists else None; db.commit(); db.close()"
```

激活测试：

```powershell
curl.exe -X POST http://127.0.0.1:8000/license/activate -H "Content-Type: application/json" -d "{\"license_key\":\"TEST-KEY-001\",\"token\":\"登录返回的token\"}"
```

## 启动客户端

进入客户端目录：

```powershell
cd D:\JCC\jcc-s17-ai\client
```

安装依赖：

```powershell
npm install
```

启动 Vite：

```powershell
npm run dev
```

启动 Electron：

```powershell
npm run electron:dev
```

## 打包 Windows 安装包

进入客户端目录：

```powershell
cd D:\JCC\jcc-s17-ai\client
```

执行打包：

```powershell
npm run dist:win
```

安装包输出目录：

```text
D:\JCC\jcc-s17-ai\client\dist-release
```

应用名称：

```text
JCC S17 AI Assistant
```

打包目标：

